# Loan Portfolio Risk Analysis

Excel dashboard analysing default patterns across 10,000 Lending Club loans.

## Key Findings
- Grade F defaults at 41.6%, marginally higher than Grade G (40.6%) — 
  risk ladder breaks at extreme grades due to small sample size (n=69 for G)
- DTI is a strong default predictor — borrowers with DTI 30+ default 
  at nearly 3x the rate of DTI 0-10 borrowers
- Home ownership has minimal impact — renters (22%) and owners (21%) 
  default at nearly identical rates, suggesting income stability 
  matters more than asset ownership
- Mortgage borrowers are the safest segment (16% default rate) and 
  represent 50% of the total portfolio

## Tools Used
Microsoft Excel — pivot tables, COUNTIFS, AVERAGEIFS, slicers, charts

## Data Source
Lending Club loan data via Kaggle — 10,000 loan sample (2007–2018)
